diko.ml

새로운 Discord 초대 링크

diko.ml을 사용하면 기존보다 훨씬 간결한 무한 링크, 커스텀 링크를 무료로 쓸 수 있어요.

더이상 링크가 지워져서 링크를 다시 보내거나 커스텀 링크를 쓰기 위해 서버 부스트를 하지 말아요.




diko.ml의 장점

짧은 초대 링크

기존의 discord.gg/oooooo 코드 대신 훨씬 짧은 diko.ml/oooo 형식의 코드가 제공돼요

만료되지 않는 링크

봇을 추방/차단하거나 초대링크를 변경, 삭제하지 않는 한 초대링크는 유저가 diko.ml 링크로 접속할 때마다 만들어져요. 더이상 초대 링크가 만료됐다고 링크를 바꾸지 말아요.

무료로 제공되는 커스텀 링크

Discord 공식 커스텀 링크는 서버 부스트 LV 3이 돼야 제공돼요. 하지만 diko.ml은 무료로 커스텀 링크가 제공돼요. 나만의 커스텀 링크를 만들어보세요.




diko.ml 사용 방법

  1. 먼저 이 링크를 눌러 봇을 초대해요.
  2. 아무 채널에서나 !url을 입력해요. 커스텀 링크는 !url <커스텀 링크> 를 입력해요.
  3. 봇이 메세지를 보내면 ✅를 눌러요.
  4. URL 설정이 완료되었어요! 해당 url로 들어가보세요.
  5. url을 바꿀 때도 같은 방법을 쓰면 돼요.
  6. url을 지울 때는 !remove를 사용하거나 봇을 추방하면 돼요.
